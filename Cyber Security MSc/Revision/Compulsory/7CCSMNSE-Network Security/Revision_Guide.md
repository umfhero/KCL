# Network Security (7CCSMNSE) Revision Guide

**Credits:** 15 | **Semester:** Spring | **Assessment:** ~85% Exam / ~15% Coursework

## Module Overview
Building heavily on Cryptography, they will examine:
- Network fundamentals and Internet protocols (wired and wireless)
- Network attacks, countermeasures, and defence strategies
- Network security protocol evaluation

## Preparation & Prerequisites
- **Prerequisites:** They **must** complete the Cryptography module first.
- **Background Knowledge:** TCP/IP knowledge from prior study or self-directed learning is strongly recommended. The scope encompasses wireless security and protocol-level attacks.

## Reading List
- **Primary Text:** Stallings, *Cryptography and Network Security*, 7th Ed. (2017) (Approx. £50-65).

They should store their notes on TCP/IP, network defences, protocol-level attacks, and wireless networking concepts here.
