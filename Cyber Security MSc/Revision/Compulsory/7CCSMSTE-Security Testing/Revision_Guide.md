# Security Testing (7CCSMSTE) Revision Guide

**Credits:** 15 | **Semester:** Spring | **Assessment:** Likely mix of Examination and Practical Coursework

## Module Overview
This module is the most directly relevant to offensive security careers. They will cover:
- Penetration testing methodology and ethical hacking
- Social engineering
- Web application, database, and system-level attacks
- Vulnerability assessment and security testing frameworks

## Preparation & Prerequisites
- **Background Knowledge:** Networking fundamentals and web application basics are important prerequisites.
- **Practical:** They will learn to implement and report on actual penetration tests, utilizing KCL's virtual lab environment.

## Reading List
- **Primary Text:** Stuttard & Pinto, *The Web Application Hacker's Handbook*, 2nd Ed. (2011) (Approx. £25-35).
- **Free Alternatives:** 
  - OWASP Web Security Testing Guide v4.2 (owasp.org)
  - PortSwigger Web Security Academy (portswigger.net/web-security) - highly recommended for free, interactive labs covering major vulnerabilities.

This folder is ideal for storing notes on testing frameworks, vulnerability assessments, PortSwigger lab write-ups, and CTF challenges.
