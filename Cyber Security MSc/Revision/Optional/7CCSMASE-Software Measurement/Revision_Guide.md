# Software Measurement and Testing (7CCSMASE) Revision Guide

**Credits:** 15 | **Semester:** Autumn | **Assessment:** 85% Exam / 15% Coursework

## Module Overview
If chosen as an option, this analytically rigorous module focuses on software quality assurance rather than security testing. They will explore:
- Software lifecycle testing, white-box and black-box testing
- Finite models (control flow graphs, data-flow graphs, finite state machines)
- Coverage metrics, symbolic execution, static analysis
- Test oracles and automated test data generation

## Preparation & Prerequisites
- **Background Knowledge:** Programming experience and basic algorithms knowledge are recommended.

## Reading List
- **Primary Text:** Ammann & Offutt, *Introduction to Software Testing*, 2nd Ed. (2017) (Approx. £50-60).

They can use this folder to compile algorithms revision, finite model diagrams, and metric calculations before the term starts.
