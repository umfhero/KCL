# Distributed Ledgers and Crypto-Currencies (7CCSMDLC) Revision Guide

**Credits:** 15 | **Semester:** Spring | **Assessment:** Not publicly confirmed (likely includes practical elements)

## Module Overview
If chosen as an option, they will learn applied cryptography as it relates to blockchain tech:
- Peer-to-peer consensus networks
- Bitcoin architecture, Ethereum, and smart contracts
- Mechanism design, game theory, and the CAP theorem
- Permissioned vs permissionless networks

## Preparation & Prerequisites
- **Prerequisites:** Strong cryptography knowledge is a de facto prerequisite. This module is peripherally related to core cybersecurity but broadens career horizons.

## Reading List
- **Primary Text (FREE):** Narayanan et al., *Bitcoin and Cryptocurrency Technologies* (Princeton, 2016) - free draft PDF at bitcoinbook.cs.princeton.edu.
- **Alternative (FREE):** Antonopoulos's *Mastering Bitcoin* (3rd edition) - fully open-source on GitHub.

Use this folder to store smart contract practice code, blockchain fundamentals, and notes from the Princeton textbook.
